# Dark+

A faithful port of Visual Studio Code's iconic **Dark+** theme to Zed — the most-used theme in the world's most popular editor, now at home in the world's fastest editor.

Every syntax color is mapped directly from VS Code's source: yellow functions, teal types, light-blue variables, salmon strings, green comments, and more. UI colors, terminal ANSI palette, and editor chrome are all carefully matched to give you the same familiar feel without compromise.

## Installation

Search for **VSCode Dark+** in Zed's extension marketplace (`cmd+shift+p` → `zed: extensions`).

| Element | Color |

| Strings | `#CE9178` |
| Numbers | `#B5CEA8` |
| Comments | `#6A9955` |
| Regex | `#D16969` |
